3. SPÉCIFICATIONS FONCTIONNELLES
================================================================================

3.1 STRUCTURE DU SITE
----------------------
Site web mono-page (Single Page Application) organisé en 5 sections :

A) ACCUEIL
   - Présentation générale du profil
   - Localisation (Bruxelles, Belgique)
   - Statut de disponibilité (Disponible/Non disponible)
   - Niveau d'expérience (Junior)
   - Avatar ou photo de profil

B) FORMATION
   - Timeline chronologique des formations et expériences
   - Formation Web Developer (Interface3, 2025-2026)
   - Expérience Assistante Dentaire (ULB-EFP, 2019-2025)
   - CESI - Sciences (Notre Dame des Champs, 2015-2016)
   - Icône de téléchargement du CV en PDF

C) PORTFOLIO
   - Grille de projets (minimum 6 projets)
   - Pour chaque projet :
     * Titre du projet
     * Description courte
     * Technologies utilisées (tags)
     * Statut (Actif, En cours, Complété)
     * Liens vers démo et code source
   - Projets actuels :
     * Security Point (PHP, Javascript, MySQL)
     * CV en ligne (Html, css)

D) COMPÉTENCES
   - Classification par catégories :
     * Front-End : JavaScript/TypeScript, React/Vue.js, HTML5/CSS3
     * Back-End : Node.js/Express, Python/Django, SQL/MongoDB
     * DevOps : Docker, AWS/Azure
   - Barres de progression visuelles pour chaque compétence
   - Pourcentage de maîtrise affiché

E) CONTACT
   - Cartes d'informations de contact :
     * Email professionnel
     * Profil LinkedIn
     * Compte GitHub
     * Numéro de téléphone
   - Liens cliquables/fonctionnels

3.2 FONCTIONNALITÉS INTERACTIVES
----------------------------------
- Navigation fluide entre sections sans rechargement de page
- Menu de navigation avec 5 boutons :
  * Icônes visuelles pour chaque section
  * État actif/inactif visible
  * Navigation au clic

- Éléments animés flottants :
  * Vaisseau spatial avec tooltip d'informations de contact
  * Météorite avec tooltip et lien de téléchargement CV

- Effets visuels :
  * Effet glitch sur le titre principal
  * Scanline animée (effet écran vintage)
  * Bordures animées sur les cartes
  * Barres de chargement animées
  * Transitions fluides entre sections

- Modal 3D pour les démos de projets (intégration Three.js)

3.3 TÉLÉCHARGEMENT DE CV
-------------------------
- Bouton de téléchargement principal (météorite flottante)
- Icône de téléchargement secondaire dans la section Formation
- Format de fichier : PDF
- Nom du fichier : CV_Anais_Kadri.pdf
